Web App


The Sight Reading Trainer web application is a personal project for open-source music education. The purpose of the application is to generate random notes on a music sheet while the user tries to play them on a connected MIDI device, measuring its performance ratio. The notes generation is fully configurable, allowing the user to control the beats-per-second, key, signature, notes allowed, range and figures.
