Finite Element Analysis
Web App


The Finite Element Analysis web app is a personal project to develop an open-source unique application for mechanical static analysis, which is currently not existent in the market. The application allows the user to configure the geometry, parameters and load conditions on the client-side, while solving the differential equations on the server-side.
